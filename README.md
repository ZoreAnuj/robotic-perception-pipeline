# Robotic Perception Pipeline

This project implements a ROS2-based perception and motion planning pipeline for robotic pick-and-place tasks. It explores the integration of advanced computer vision with industrial robot control to enable intelligent object manipulation in simulated environments.

## Key Features
* Real-time object detection and pose estimation using YOLOv8-OBB.
* Motion planning and execution for Franka Panda and UR5 arms via MoveIt 2.
* Simulation and visualization within NVIDIA Isaac Sim for development and testing.
* Modular ROS2 Humble nodes for scalable perception and control.

## Tech Stack
ROS2 Humble, MoveIt 2, YOLOv8-OBB, NVIDIA Isaac Sim, Python, C++
